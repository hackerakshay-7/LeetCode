Ceil The Floor

Difficulty: EasyAccuracy: 43.76%Submissions: 187K+Points: 2

Given an unsorted array arr[] of integers and an integer x, find the floor and ceiling of x in arr[].

Floor of x is the largest element which is smaller than or equal to x. Floor of x doesn’t exist if x is smaller than smallest element of arr[].
Ceil of x is the smallest element which is greater than or equal to x. Ceil of x doesn’t exist if x is greater than greatest element of arr[].

Return an array of integers denoting the [floor, ceil]. Return -1 for floor or ceiling if the floor or ceiling is not present.

Examples:

Input: x = 7 , arr[] = [5, 6, 8, 9, 6, 5, 5, 6]
Output: 6, 8
Explanation: Floor of 7 is 6 and ceil of 7 is 8.

Input: x = 10 , arr[] = [5, 6, 8, 8, 6, 5, 5, 6]
Output: 8, -1
Explanation: Floor of 10 is 8 but ceil of 10 is not possible.

Expected Time Complexity: O(n)
Expected Auxiliary Space: O(1)

Constraints :
1 ≤ arr.size ≤ 105
1 ≤ arr[i], x ≤ 106

class Solution {
    public static int ff(int[] arr,int n,int x){
        //************BINARY SEARCH APPROACH *************************
        int low=0;
        int high=n-1;
        int ans=-1;
        while(low<=high){
            int mid=(low+high)/2;
            if(arr[mid]<=x){
                ans=arr[mid];
                low=mid+1;
            }
            else{
            high=mid-1;
            }
            
        }
        return ans;
    }
    
    public static int cc(int arr[],int n,int x){
        int low=0;
        int high=n-1;
        int ans=-1;
     while(low<=high){
         int mid=(low+high)/2;
         if(arr[mid]>=x){
             ans=arr[mid];
             high=mid-1;
         }
         else{
             low=mid+1;
         }
     }   
     return ans;
    }
    public int[] getFloorAndCeil(int x, int[] arr) {
        // code here
        int n=arr.length;
        Arrays.sort(arr);
        int floor=ff(arr,n,x);
        int ceil=cc(arr,n,x);
        return new int []{floor,ceil};
        
    }
}

SIMPLE JAVA 

class Solution {
    public int[] getFloorAndCeil(int x, int[] arr) {
        Arrays.sort(arr); // Ensure array is sorted

        int floor = -1;
        int ceil = -1;

        int low = 0, high = arr.length - 1;

        // Find floor
        while (low <= high) {
            int mid = low + (high - low) / 2;

            if (arr[mid] <= x) {
                floor = arr[mid]; // possible floor
                low = mid + 1;     // move right to find greater but still <= x
            } else {
                high = mid - 1;
            }
        }

        // Reset for ceil
        low = 0;
        high = arr.length - 1;

        // Find ceil
        while (low <= high) {
            int mid = low + (high - low) / 2;

            if (arr[mid] >= x) {
                ceil = arr[mid]; // possible ceil
                high = mid - 1;   // move left to find smaller but still ≥ x
            } else {
                low = mid + 1;
            }
        }

        return new int[]{floor, ceil};
    }
}

    def getFloorAndCeil(self, x: int, arr: list) -> list:
        
        # here we have to return both upper and lower bound
        floor = -1
        ceil = -1
        n = len(arr)
        
        for element in arr:
            # case where we have found the target 
            # where both floor and ceil will be same
            
            if element == x:
                floor = element
                ceil = element
                return [floor, ceil]
                
            # getting the floor, means less than the target value
            # element > floor means we want the largest value less than target
            
            if element  < x and element> floor:
                
                floor = element
                
                
            # getting the ceil means greater than the target

            # if the ceil is -1 it will update to element and element<ceil means we want the smallest number greater than the element

 

            if element > target and (ceil  == -1 or element < ceil)
                
                ceil = target
                
                
        return [floor, ceil]

Python - Easy To Understand | Time - O(n) | Space - O(1)

    def getFloorAndCeil(self, x: int, arr: list) -> list:
        floorVal = -1         # initial val
        ceilIVal = -1
        floorDiff = float("inf")    # difference between target and arr[i]
        ceilDiff = float("inf")
        for i in range(len(arr)):
            fmin = x - arr[i]
            if fmin >=0 and fmin < floorDiff:
                floorVal = arr[i]
                floorDiff = fmin
            cmin = arr[i] - x 
            if cmin >=0 and cmin < ceilDiff:
                ceilIVal = arr[i]
                ceilDiff = cmin
                
        return floorVal, ceilIVal
